Introduction

Database (database) is not a new term, but this concept belongs to any system. What is a system? The system is simply like a machine that has an input to receive data, a set of computations (processing unit) that performs computational operations on the data, and finally gives us an expected output. The system can be like the complex structure of the human brain, with sensors (five senses) each receiving input and sending it to the information processing center or brain, and the brain producing the appropriate response (output).
